--Sistema de Nómina--

Este programa está basado en la necesidad de la creación de un entorno de gestión de nóminas 
para diversos empleados, tales como empledaos por horas, empleados por comisión o empleados por
ventas. Ademas de esto, permite la recopilación de la información del sistema para generar un 
archivo que permita llevar un registro semanal de todas las nóminas de los empleados.
